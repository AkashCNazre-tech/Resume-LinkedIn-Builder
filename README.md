📄 Resume & LinkedIn Profile Builder
An AI-powered web application that helps users generate a professional resume and get personalized LinkedIn profile improvement suggestions — all in one place.
> Built with HTML, CSS, Vanilla JavaScript, and the Anthropic Claude AI API.

✨ Features
AI-Powered Resume Generator — Fills in your details and the AI polishes it into a professional, recruiter-ready resume
LinkedIn Profile Analyser — Scores your LinkedIn profile (0–100) and gives section-by-section improvement tips
Copy to Clipboard — Copy your generated resume text in one click
Responsive Design — Works on desktop and mobile
Real-time Feedback — Instant AI suggestions for headline, about section, skills, experience, and more

⚙️ Setup & Usage
1. Clone the repository
```bash
git clone https://github.com/akashnazare/resume-linkedin-builder.git
cd resume-linkedin-builder
```
2. Open in browser
Since this is a pure HTML project, simply open `index.html` in your browser:
```bash
open index.html
```
Or use a local server (recommended):
```bash
# Using VS Code Live Server extension
# OR
npx serve .
```
3. Use the app
Fill in your personal info, skills, experience, education, and projects
Click "Generate Resume & LinkedIn Tips"
View your polished resume under the Resume Preview tab
See your LinkedIn improvement score and tips under the LinkedIn Improvements tab
---
🔑 API
This project uses the Anthropic Claude API (`claude-sonnet-4-20250514` model) to:
Generate polished, professional resume content
Score and analyse LinkedIn profile sections
Provide actionable, personalised career tips
---
📋 What the LinkedIn Analyser Checks
Section	What it looks for
Profile Photo	Presence and quality guidance
Headline	Keywords, role clarity, value proposition
About / Summary	Completeness, tone, call to action
Experience	Achievement-focused bullets, dates
Skills	Relevance and number of skills
Recommendations	Social proof
Featured Section	Portfolio and pinned work
Custom URL	Personal branding
